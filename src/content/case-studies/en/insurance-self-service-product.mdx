## Context

I joined an existing insurance product rather than starting a new one. The main value was not greenfield speed, but improving a codebase that already had operational weight.

## My Role

- owned the frontend alone
- supported ongoing product work
- handled maintenance and refactoring in parallel

## Technical Work

- extracted reusable components from page-level implementations
- separated the API layer from page code
- improved state storage and reduced complexity around data handling
- cleaned up the codebase in a noticeable, not merely cosmetic, way

## Outcome

This project is a good example of senior-level maintenance work: less about flashy launches, more about making an existing product calmer, smaller, and easier to change without breaking it.
