## Context

This product had an unusual advantage: it was paused. That removed a lot of rollout pressure and made it possible to do larger technical work more cleanly than in an always-on production stream.

## My Role

- worked as the senior frontend engineer alongside one middle-level developer
- contributed to architecture decisions
- helped define the new frontend direction

## Technical Changes

- migrated the codebase from Vue 2 to Vue 3
- separated frontend work from the monolithic structure
- moved data access behind dedicated APIs
- contributed to testing setup, code review, and mentoring

## Outcome

Because the product was paused, the team had time to rebuild deliberately instead of patching in place. That made this project valuable as a modernization case: not a heroic hotfix story, but a controlled technical reset.
